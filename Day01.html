HTML공부하기

HTML은 웹 페이지를 만들 때 사용하는 마크업 언어로, 웹 컨텐츠의 구조와 내용을 정의하는데 사용됩니다.
HTML은 웹 브라우저가 이해하고 해석할 수 있는 형식으로 웹 페이지를 작성하는 데 필수적인 언어입니다.
Mac 사용시 실행 단축키 (fn + control + f5)

용어 설명
HTML 문서 구조 HTML 문서는 <html> 요소로 시작해서 </html> 요소로 끝납니다. 모든 HTML 코드는 이러한 태그 사이에 들어갑니다.
요소(Elements) HTML 문서의 구성 요소를 나타냅니다. 모든 요소는 시작 태그와 종료 태그로 둘러싸여 있으며, 종료 태그는 이름 앞에 슬래시(/)를 붙여 구분합니다. 예를 들어, <p>는 단락 요소를 나타내고, </p>는 이 단락 요소의 종료를 나타냅니다.
태그(Tags) HTML 요소를 정의하기 위해 사용되는 것으로, < > 꺾쇠 괄호 안에 태그 이름이 들어갑니다. 예를 들어, <p>는 단락을 시작하는 태그입니다.
속성(Attributes) HTML 요소에 추가 정보를 제공하는데 사용됩니다. 태그에 속성을 추가하면 해당 요소의 동작이나 모양을 조정할 수 있습니다. 예를 들어, <img> 요소에는 src(이미지 경로)와 alt(대체 텍스트)와 같은 속성이 있습니다.
텍스트 내용 HTML 문서는 텍스트 내용을 포함합니다. 텍스트는 단순한 문자열 또는 다른 HTML 요소일 수 있습니다.
주석(Comments) HTML 문서 내에 주석을 추가하여 코드를 설명하거나 임시로 비활성화할 수 있습니다. 주석은 <!--로 시작하고 -->로 종료됩니다.
