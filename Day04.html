static 기본값, 별도의 위치를 지정하지 않고 흐름에 맞게 배치한다.

fixed 기준을 중심으로 자신만의 위치를 가진다. 스크롤의 영향을 받지 않는다
sticky 평소에는 relative처럼 배치되지만, 기준점에 도달하면 fixed와 같다

display 값 특징
inline content만큼의 크기를 차지한다. width, height를 적용할 수 없다
block 한 줄을 혼자서 차지한다. width, height를 적용할 수 있다
inline-block inline처럼 한줄에 여러요소를 배치할수 있고, 크기를 지정할 수 있다
flex 내부 요소를 일정한 기준에 따라 정렬한다
none 요소가 화면에서 사라진다. 자리차지도 없다
