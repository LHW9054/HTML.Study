02-style.html

단위
크기
  • % : 백분율
  • em : 배수
  • px : 픽셀
색상
  • HEX : #000000
  • rgb : rbg(255, 128, 0)
    red, green, blue
  • rgba : rgba(255,255,192,0.5)
    red, green, blue, alpha
  • colorName : lightgreen
가시 속성
display 화면에 보이는 형태지정
visivility 보이거나 보이지 않거나
opacity 투명도 조절
박스속성
width, height 영역의 어비와 높이를 지정
margin, padding 테두리 바깥 여백, 테두리 안쪽 여백
10px (all)
10px auto (top-bottom, left-right)
10px 5px 5px 10px (top, right, bottom, right)
box-sizing center-box : 영역의 크기는 가운데만 적용
border-box : 테두리를 포함한 영역을 적용
테두리 속성
border: 두께 유형 색상 테두리를 지정
border-radius 테두리의 둥글기를 지정
margin, padding과 같은 방식으로 상하좌우 지정가능

05-overflow.html

overflow: auto 내용에 따라서 스크롤바가 생길수도 있고 없을 수도 있다.
overflow: hidden 범위를 벗어나는 내용은 숨긴다. 스크롤바는 강제로 사라진다.
overflow: scroll 내용에 상관없이 항상 스크롤바를 표시한다.
overflow: 내용이 영역을 벗어날 경우의 처리방식을 결정
overflow-x: 내용이 가로영역을 벗어날 경우의 처리방식을 결정
overflow-y: 내용이 세로영역을 벗어날 경우의 처리방식을 결정
