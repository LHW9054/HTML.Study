02-input.html

사용자의 입력밧을 서버에 전송하기 위한 요소. form과 연계하여 사용한다.

태그 type 특징 예시
input text 일반적인 텍스트를 입력받는다.
password 마스킹된 텍스트를 입력받는다
number 숫자를 입력받는다
date 날짜를 입력받는다
range 마우스로 범위사이의 숫자를 입력받는다
file 사용자의 첨부파일을 입력받는다
radio 여러요소 중 하나를 선택하여 입력받는다
checkbox 여러 요소중 해당하는 값을 복수 선택하여 입력받는다
hidden 이름과 값이 정해져 있으나, 시각적으로 보이지 않는다(display: none)
color 색상값을 입력받는다
select 여러 선택지 중 하나를 입력 받는다 (radio 와 방식이 유사하다)
textarea 여러 줄로 구성된 텍스트를 입력받는다
